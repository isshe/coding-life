

# 总体设计
```
+------------+             +------------+            +------------+
| 协议解析器 | (n)--->(n) | 中继服务器 | (n)--->(n) | 代理服务器 |
+------------+             +------------+            +------------+
```
* 协议解析器：用于解析socks5等协议。
* 中继服务器：只是转发流量。
* 代理服务器：进行代理。
* 实现设想：
  * 支持多入口多出口多协议。
  * 协议解析器模块化，支持多种协议。（入口）
  * 代理服务器支持多协议。（TCP/UDP，出口）
  * 实现成到一个框架内，各种组件只是充当不同角色，实现方式类似。
