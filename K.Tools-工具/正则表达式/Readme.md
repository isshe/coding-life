# 邮箱地址
> 字母、数字、下划线、`.`

```
^[.0-9a-zA-Z_]{1,18}@([0-9a-zA-Z-]{1,13}\.){1,}[a-zA-Z]{1,3}$
```

# 用户名
> 字母、数字、下划线、6-12个字符

```
^[0-9a-zA-Z_]{6,12}$
```

# 手机号
> 1开头，11位

```
^1\d{10}$
```

# 中文字符
```
[\u4e00-\u9fa5]
```
