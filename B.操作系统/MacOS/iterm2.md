# 单词跳转
* 向前跳转单词
  * 快捷键：`options + 左`
  * 设置步骤：
    * `+`
    * Action: `Send Escape Sequence`
    * ESC+: `b`

* 向后跳转单词
  * 快捷键：`options + 右`
  * 设置步骤：
    * `+`
    * Action: `Send Escape Sequence`
    * ESC+: `f`

* 向前删除单词
  * 快捷键：`options + 上`
  * 设置步骤：
    * `+`
    * Action: `Send Hex Code`
    * `0x17`
