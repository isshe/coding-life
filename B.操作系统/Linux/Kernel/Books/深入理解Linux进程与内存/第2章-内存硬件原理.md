## 查看内存信息

```bash
cat /proc/meminfo

dmidecode | grep -A16 'Memory Device'
```

## 核心频率

核心频率是内存电路的震荡频率，是内存一下工作的基石。

## DIMM 是什么？DIMM 分类？

DIMM：Dual In-Line Memory Module，双列直插式内存模块。
