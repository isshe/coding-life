# Linux 系统存储空间占用情况分析

此文档记录：

- 分析 Linux 系统存储空间占用的方法
- 排查磁盘空间泄漏的方法

##  使用 du 命令逐级检查磁盘占用情况

```
$ sudo du -hs /*
...
0	/bin
165M	/boot
8.0K	/cores
0	/dev
5.6M	/etc
662M	/home
...
```

从根目录开始逐级排查占用空间大的目录，确认不再需要后，即可进行清理。


##

```
find . -maxdepth 1 -type d -not -path "." -not -path "./lost+found" -not -path "./dev" -not -path "./proc"  | xargs -n1 du -hs | sort -hk1

lsof -P | grep deleted > log.txt
```
