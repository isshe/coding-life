题目地址：https://leetcode.com/problems/convert-sorted-list-to-binary-search-tree/    
题目描述：给一个排好序的链表，构造一棵平衡的二叉查找树。  
思路：每次用中间节点做根并且中间节点把链表分为left和right两部分。这两部分再分别重复以上操作。  
方法一：时间复杂度O(n^2), 空间复杂度O(1), 并且修改了链表。  
1. 取链表长度， （每次递归都要取，耗时间）  
2. 遍历链表得到中间元素，并把链表分为两部分。（也耗时间）  
3. 递归。  
方法二：利用BST中序遍历结果已排序的特点， 使用中序递归。  
		时间复杂度O(n), 空间复杂度O(1),不修改链表.    

1. 递归前，取链表长度。  
2. 进行中序递归。
