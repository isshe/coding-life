## 基础
* 使用局部变量有什么好处？
    * 可以避免不必要的命名而造成全局变量的混乱；
    * 可以避免同依程序中不同代码部分的命名冲突；
    * 访问局部变量更快；
    * 局部变量能及时回收释放；
* 为什么Lua语言会限制goto语句不能跳出一个函数。（提示：你要如何实现这个功能？）
* 什么是高阶函数？
    * 以函数为参数的函数。
* 什么是`非局部变量`？
    * 被函数中的匿名函数所访问的函数中的局部变量，被称为非局部变量。
    * 例如：下面代码中，count就是非局部变量。
    ```lua
    function newCounter()
        local count = 0
        return function()
            count = count + 1
            return count
       end
   end
    ```
